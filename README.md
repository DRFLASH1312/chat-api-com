# Chat Api Com

## Installation & Usage
### Composer

To install the bindings via [Composer](http://getcomposer.org/), add the following to `composer.json`:

```
{
  "repositories": [
    {
      "type": "git",
      "url": "https://github.com/drdim/chat-api-com.git"
    }
  ],
  "require": {
    "drdim/chat-api-com": "*@dev"
  }

```
